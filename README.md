# CenterPose

## Overview

This repository is customized version of the repository(https://github.com/NVlabs/CenterPose) which is the official implementation of the paper [Single-Stage Keypoint-based Category-level Object Pose Estimation from an RGB Image](https://arxiv.org/abs/2109.06161) by Lin et al., ICRA 2022 (full citation below).
